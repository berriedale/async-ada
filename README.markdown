async-ada
==========


The **async-ada** project's goal is to wrap some of the fabulous
platform-specific asynchronous I/O APIs available, namely:

  * `epoll(7)` on Linux 2.6+
  * `kqueue(2)` on BSD/Mac OS X systems


Current Status
---------------

Unusable
